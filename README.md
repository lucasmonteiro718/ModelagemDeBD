# ModelagemDeBD
Visão Geral e Propósito do Minimundo

O sistema tem como objetivo gerenciar o empréstimo e a consulta de livros digitais por usuários
cadastrados. Seu público-alvo são leitores que utilizam bibliotecas digitais e bibliotecários
responsáveis pelo acervo. O propósito é facilitar o acesso, organização e controle de livros digitais
de forma eficiente.
